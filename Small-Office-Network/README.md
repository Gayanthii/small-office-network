**Project Title**

Small Office Network Design using Cisco Packet Tracer



**Project Description**

This project demonstrates the design and implementation of a small office network using Cisco Packet Tracer. It includes multiple LANs connected through a router, enabling communication between different networks. The project also includes DHCP configuration for automatic IP assignment.



**Network Topology**

\- LAN 1: 192.168.1.0/24

\- LAN 2: 192.168.2.0/24

\- Devices: Router, Switches, PCs



**Features**

\- LAN setup using switches

\- Router-based inter-network routing

\- Static and dynamic IP addressing

\- DHCP configuration for automatic IP assignment

\- Connectivity testing using ping

\- Web server configuration (HTTP service) for client-server communication

\- Packet flow analysis using Simulation Mode



**Technology Used**

\- Cisco Packet Tracer

\- Networking concepts (IP addressing, subnetting, routing, DHCP)



**How it Works**

\- PCs in LAN 1 and LAN 2 are connected via switches

\- Router connects both LANs and enables communication between them

\- DHCP is configured on the router to assign IP addresses automatically

\- Connectivity is verified using ping commands



**Testing Results**

\- Successful communication between LAN 1 and LAN 2 using ping  

\- DHCP successfully assigned IP addresses automatically  

\- Web server accessed successfully from client PCs  

\- Router correctly forwarded packets between networks
